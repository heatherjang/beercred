#BeerCred
<h4>Preview this app <a href="http://beercred.herokuapp.com/">here!</a></h4>
<p>BeerCred is a demo app that allows users to electronically purchase beer from the Launch Academy beerfridge. Users with admin status can view all user activity and update inventory count. As this app was built for demo purposes only, it is still under development.</p>
<h4>Technology Used</h4>
<ul>
<li>Ruby</li>
<li>HTML/CSS</li>
<li>SQLite3 (Development)/PostgreSQL(Production)</li>
<li>Sinatra</li>
<li><a href="https://stripe.com/docs/api">Stripe API</a></li>
</ul>
